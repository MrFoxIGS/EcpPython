Palindromes

Palindromes are words that are the same forwards and backwards. e.g. radar, wow and noon

In this first task you will be creating a function palindrome that has a parameter of a string which is a lower case word. The function will return a boolean, True if the word is a palindrome and False if the word isn't a palindrome

e.g. palindrome('kyak') will return True, and palindrome('canoe') will return False

You can use the list() function to convert a string to a list
