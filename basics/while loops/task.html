While Loops

The while loop will continue to execute the commands within the loop while the condition is met. In this case while the user input is not equal to q

The synax for the while loop is as follows, note that the commands to be executed inside the loop are indented

The while loop will keep executing indefinately until the condition is met

The break command will break out of a while loop prematurely

Loop counters keep track of how many times the loop has run. += and -= increase and decrease the loop counter.
