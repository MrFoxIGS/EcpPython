Squares

In this task you will be writing 3 functions and using functions within other functions

Square of sum

This function has a parameter of an integer - number and returns an integer that is the square of the sum of all the integers up to number
e.g. square_of_sum(3) will return 36, (1+2+3)^2

Sum of sqaures

The function has a parameter of an integer -number and returns an integer that is the sum of all of the square numbers up to number
e.g. sum_of_squares(3) will return 14, 1^2+4^2+9^2

Difference of squares

The final function has a parameter of an integer - number and will return the difference between the functions square_of_sums(number) and sum_of_squares(number)
e.g. difference_of_squares(3) returns 22, 36-14
