DNA Complimentary Base Pairs

Complementary base pairing is the phenomenon where in DNA guanine(G) always hydrogen bonds to cytosine(C) and adenine(A) always binds to thymine(T).

You will need to create a function to_comp, which has a parameter of a string of bases (dna_strand) and returns a string of complimentary bases (new_strand)

e.g. to_comp('GATTACA') would return 'CTAATGT'
